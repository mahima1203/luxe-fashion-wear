# Luke Fashion Wear

Welcome to **Luke Fashion Wear**, your destination for premium fashion. Hand-curating the world's best brands since 2026.

This project is a modern ecommerce frontend built with **Next.js App Router**, featuring dynamic routing, responsive design, and seamless user experiences.

## ✨ Features
- **Dynamic Storefront**: Browse curated trending products and new arrivals.
- **Category Browsing**: Dedicated pages for seamless product discovery across categories.
- **Responsive Design**: fully optimized for desktop, tablet, and mobile devices using Tailwind CSS.
- **Next.js App Router**: Built using the latest Next.js 15 features for optimal performance and SEO.
- **Authentication**: Integrated NextAuth for secure user dashboards (currently optional).

## 🚀 Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/mahima1203/luxe-fashion-wear.git
   cd luxe-fashion-wear
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

## 🛠️ Tech Stack
- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Icons**: Heroicons
- **Database**: PostgreSQL (Vercel Postgres)
- **Deployment**: Vercel ready
